Per utilizzare il gestionale Team Management, è necessario innanzitutto creare il database con mysql
workbench tramite le query di creazione del database che si trovano all'interno del file GSS.ddl.
Successivamente sarà necessario popolare alcune tabelle, tramite il file Popolazione_tabelle.sql.
La classe da lanciare per avviare il gestionale è Launcher.java situata all'interno del package main;
All'avvio del programma verrà richiesto l'inserimento della propria password di root di mysql workbench.
